# 数据存储单元
存储数据时，我们可以使用不同的单位来衡量数据的大小。

在数据计量单位中，大写字母“B”表示“字节”，小写字母“b”表示“位”。
字节 (B) 千字节 (kB) 兆字节 (MB) 千兆字节 (GB) 太字节（TB）
在存储数据时，我们使用这些单位来衡量数据的大小。

如，存储下图所示的 500x300 像素的老虎图像，每个像素用 3 个 字节存储颜色（24 位 颜色深度），则需要 500 * 300 * 3 = 450 000 字节。这张图大小就是 450000 字节，即 450 kB（千字节）。

 1 千字节（kB）有时指的是 2 ^10 = 1024 字节而不是 1000 字节，使用这种测量方法，上面的图像大小为 450 000 / 1024 = 441kB
为避免这种混淆，kibi-（Ki），意为 1024，mebi-（Mi），意思是1024*1024，gibi-（Gi），意思是1024*1024*1024为了绝对清楚和正确，我们应该说上面的图片要么是 450 kB，要么是 441 KiB。
# 数据传输单元
测量数据传输速度就像测量数据存储单位一样，只是最后要加上“每秒”的单位。

数据传输速度以比特为单位的单位是“比特每秒”（bps 或 b/s），数据传输速度以字节为单位的单位是“字节每秒”（Bps 或 B/s）。
当传输大量数据时，我们会使用前缀：
千比特每秒（kbps）
兆比特每秒 (Mbps)
千兆比特每秒 (Gbps)
其中，kilo-、mega- 和 giga- 分别指 1000、1000000 和 1000000000。
